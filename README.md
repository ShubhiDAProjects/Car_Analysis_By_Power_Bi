# 🚗 Car Sales Analysis | Power BI Project

## 📌 Project Overview

This project focuses on analyzing car sales data to understand market trends, brand performance, and customer preferences. An interactive Power BI dashboard is developed to visualize key insights and support business decision-making.

---

## 🎯 Objectives

* Analyze total car sales and revenue trends
* Identify top-performing car brands and models
* Understand pricing patterns and demand
* Evaluate sales by region, category, or fuel type
* Provide insights for improving sales strategy

---

## 🛠️ Tools & Technologies Used

* **Power BI** → Data visualization & dashboard development

---

## 🔄 Project Workflow

### 1. Data Preparation

* Imported dataset into Power BI
* Cleaned and transformed data using Power Query
* Handled missing values and standardized data

---

### 2. Data Modeling

* Built relationships between tables (if applicable)
* Created calculated columns and measures using DAX

---

### 3. Dashboard Development

* KPI Cards (Total Sales, Total Revenue, Avg Price, Units Sold)
* Bar charts for brand and model performance
* Line charts for sales trends over time
* Pie/Donut charts for fuel type or category distribution
* Slicers for filtering (brand, region, fuel type)

---

## 📊 Key Insights

* Identified top-selling car brands and models
* Observed pricing trends and their impact on demand
* Certain categories (e.g., SUV, Sedan) dominate sales
* Regional differences affect sales performance

---

## 📈 Outcome

* Built a clear and interactive dashboard
* Enabled better understanding of market trends
* Demonstrated Power BI and data visualization expertise

---
